# 1. Project Title
Retail Sales Analytics Dashboard 

## 2. Short Description
Strategic analytics solution providing deep insights into retail sales dynamics across regions, channels, and customer segments—monitoring total sales (19K units), channel performance (Online 50.9% vs Offline 49.1%), and acquisition metrics (43% new customers)—to optimize inventory, pricing, and regional expansion strategies.

## 3. Tech Stack
ETL & Processing: Power Query for advanced data cleaning/transformation  
Database & Querying: MySQL, SQL queries/aggregations  
Visualization & Modeling: Power BI (DAX measures for sales/channel calculations, star schema data modeling, slicers/maps/filters, dynamic relationships)  
Version Control: Git/GitHub

## 4. Data Source
Retail sales transactions pulled directly from MySQL database (regional sales, customer types, timestamps), transformed via Power Query for normalization and enrichment, then semantically modeled in Power BI for interactive exploration.

## 5. Features and Highlights
Core KPIs: Total sales volume (19K), geographic splits (East/North/South/West), channel contribution pie charts, monthly peaks (Sep at 1.8K units)  
Visual Analytics: Regional heatmaps, stacked column charts for time trends, customer funnel visuals (43% new/57% returning), cross-highlighting across dimensions  
Advanced Insights: Performance benchmarking by region/channel, trend forecasting indicators, top/bottom category analysis, mobile-optimized drill-downs—enhancing decision velocity by 35-50%  
Scalability: Enterprise-grade data refresh, complex DAX for custom metrics (sales velocity/growth rates), RLS-ready security framework
